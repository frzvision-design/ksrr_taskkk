# Logo Setup / إعداد الشعار

## English
Place your company logo file named `logo.jpg` in this directory.

**Requirements:**
- File name: `logo.jpg`
- Recommended size: 512x512 pixels or larger
- Format: JPG/JPEG
- The logo will be displayed in a circular frame on the login screen

## Arabic / العربية
ضع ملف شعار شركتك باسم `logo.jpg` في هذا المجلد.

**المتطلبات:**
- اسم الملف: `logo.jpg`
- الحجم الموصى به: 512×512 بكسل أو أكبر
- التنسيق: JPG/JPEG
- سيتم عرض الشعار في إطار دائري على شاشة تسجيل الدخول

## Persian / فارسی
فایل لوگوی شرکت خود را با نام `logo.jpg` در این پوشه قرار دهید.

**الزامات:**
- نام فایل: `logo.jpg`
- اندازه پیشنهادی: 512×512 پیکسل یا بیشتر
- فرمت: JPG/JPEG
- لوگو در یک فریم دایره‌ای در صفحه لاگین نمایش داده می‌شود
